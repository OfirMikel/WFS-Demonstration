Algorithm

The Wavefunction Collapse Algorithm teaches your computer how to riff. The algorithm takes in an archetypical input, and produces procedurally-generated outputs that look like it.


In programming, the Wave Function Collapse Algorithm is a versatile tool for generating complex patterns based on an example input. It doesn't rely on neural networks or complex machine learning methods, making it an intuitive choice for those seeking clear and effective procedural generation. The algorithm works by collapsing a wavefunction that represents potential choices, with each choice affecting the entire output.
